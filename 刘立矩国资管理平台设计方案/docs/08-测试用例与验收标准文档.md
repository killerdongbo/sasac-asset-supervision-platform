# 企业资产管理系统 — 测试用例与验收标准文档

> 文档版本：v1.0  
> 更新时间：2026-04-27  
> 作者：知微

---

## 1. 文档目的

本文档用于指导开发完成后的功能测试、联调测试、客户演示验收，确保系统不是“代码写完”，而是业务流程真正可用。

---

## 2. 测试范围

| 阶段 | 测试范围 |
|---|---|
| Phase 1 | 登录、权限、用户、角色、部门、资产分类、位置、资产登记、资产清单、资产详情 |
| Phase 2 | 巡检任务、巡检执行、巡检异常、盘点任务、盘点执行、差异报告、差异调整 |
| Phase 3 | 资产流转、采购、维保维修、审批 |
| Phase 4 | 折旧、报表、预警、导入导出、部署运维 |

## 12. 最低上线标准

最低上线/演示必须满足：

1. 登录认证稳定。
2. 权限菜单基本可用。
3. 资产基础数据可维护。
4. 巡检任务可完整跑通。
5. 盘点任务可完整跑通。
6. 差异报告准确。
7. 核心操作有记录。
8. 页面无明显样式错乱。
9. 服务可稳定重启。
10. 数据库有备份方案。

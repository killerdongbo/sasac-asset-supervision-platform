#### 资产生命周期管理

- [ ] **后端：生命周期事件表** — `V17__asset_lifecycle_event.sql`
  ```sql
  CREATE TABLE asset_lifecycle_event (
      id BIGSERIAL PRIMARY KEY,
      tenant_id BIGINT NOT NULL,
      asset_id BIGINT NOT NULL,
      event_type VARCHAR(30) NOT NULL,
      event_title VARCHAR(200) NOT NULL,
      event_detail JSONB,
      source_table VARCHAR(50),
      source_id BIGINT,
      operator_id BIGINT,
      operator_name VARCHAR(50),
      event_time TIMESTAMP NOT NULL,
      created_at TIMESTAMP DEFAULT NOW()
  );
  CREATE INDEX idx_lifecycle_asset ON asset_lifecycle_event(tenant_id, asset_id, event_time DESC);
  ```

- [ ] **后端：AssetLifecycleService** — `server/src/main/java/com/sasac/platform/asset/service/AssetLifecycleService.java`

- [ ] **后端：AssetLifecycleListener** — Spring Event 监听器

#### 数据导出基础

- [ ] **后端：导出任务表** — `V18__export_task.sql`
  ```sql
  CREATE TABLE export_task (
      id BIGSERIAL PRIMARY KEY,
      tenant_id BIGINT NOT NULL,
      export_type VARCHAR(50) NOT NULL,
      file_name VARCHAR(200),
      file_path VARCHAR(500),
      status VARCHAR(20) DEFAULT 'PENDING',
      params JSONB,
      total_rows INT,
      error_message TEXT,
      created_by BIGINT,
  ```

- [ ] **后端：OrganizationService 增强** — 递归查询组织树

  - `GET /api/statistics/assets` — 穿透式资产汇总

    - 逻辑：查询 orgId + 所有下属组织的资产聚合

  - 大集团优化：>50家下属时启用 stats_cache 表预计算

- [ ] **后端：stats_cache 预计算** — `V20__stats_cache.sql`
  ```sql
  CREATE TABLE stats_cache (
      id BIGSERIAL PRIMARY KEY,
      tenant_id BIGINT NOT NULL,
      org_id BIGINT NOT NULL,
      cache_type VARCHAR(30) NOT NULL,
      cache_data JSONB NOT NULL,
      calculated_at TIMESTAMP NOT NULL,
      UNIQUE(tenant_id, org_id, cache_type)
  );
  ```
  - 定时任务：每日凌晨2点预计算各组织汇总数据

- [ ] **前端：AssetStats.vue 增强**

## 九、风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| 组织树层级过深导致递归查询慢 | 穿透统计响应超时 | 预计算+缓存，限制最大查询深度10层 |
| 历史数据无生命周期事件 | 时间线不完整 | 提供数据补录脚本，从历史业务表回填 |
| 财务系统接口变更 | 同步中断 | 适配器隔离，单系统问题不影响其他 |
| 信创数据库SQL兼容问题 | 迁移工作量大 | 提前建立SQL方言测试矩阵，逐条验证 |
| AI模型API不稳定 | 估值功能不可用 | 降级为规则引擎，AI结果仅供参考 |
